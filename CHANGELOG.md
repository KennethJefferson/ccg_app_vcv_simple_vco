## [2.0.0] - 2025-01-19

### Added
- Initial release of Tony VCO plugin for VCV Rack 2.x
- **Simple VCO** module with:
  - FREQ knob for coarse frequency control (-3 to +3 octaves)
  - FINE knob for fine tuning (-6 to +6 semitones)
  - FM input with bipolar attenuverter
  - V/OCT input with 1V/octave tracking
  - Four simultaneous waveform outputs: Sine, Triangle, Saw, Square
  - All outputs at 10Vpp level
